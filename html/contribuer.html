Pour contribuer

✦

Le projet est ouvert aux contributions. Deux modes de participation sont possibles : une contribution libre via formulaire et une contribution directe via un tableur Grist pour les utilisateurs autorisés.

Contribution libre

Accéder au formulaire de contribution →

  • Crédit contribution : indiquer le prénom et le nom du·de la contributeur·rice sous la forme Prénom Nom (majuscules à l'initiale).
  • Langue : sélectionner Latin ou Grec.
  • Auteur : renseigner le nom de l'auteur en français si existant (ex. Aulu-Gelle et non Gellius). Sinon, renseigner le nom de l'auteur en latin.
  • Œuvre : renseigner le titre de l'œuvre en français si existant. Sinon, renseigner le nom de l'œuvre dans la langue source.
  • Référence : entrer les références du texte en utilisant des chiffres arabes. Le séparateur d'usage est le point. Ne pas utiliser les notations p., § ou v. (ex. 1.114-1.138 ou 2.56).
  • Nombre de mots : indiquer le nombre de mots approximatif du texte. Ne pas spécifier « environ ».
  • Coupes : si le texte proposé a subi des coupes, cocher la case.
  • Niveau : sélectionner le niveau du texte (voir la documentation relative).
  • Titre du texte : indiquer un titre pour le texte proposé.
  • Conjugaison / Morphologie / Syntaxe : renseigner les points remarquables que présente le texte. Une liste d'items est proposée. Si un item souhaité n'apparaît pas, le renseigner dans la case « Items à renseigner ».
  • Texte latin / grec : renseigner le texte original. En prose, ne pas mettre de sauts de paragraphe ; en poésie, utiliser un saut de paragraphe entre les vers (sans séparateur /).
  • Traduction : renseigner la traduction. Même règles que pour le texte original (prose / poésie).
  • Crédit traduction : renseigner le prénom et le nom du·de la traducteur·rice sous la forme Prénom Nom.
  • Thématiques : indiquer une ou plusieurs thématiques. Séparateur : saut de paragraphe.
  • Items à renseigner : indiquer, au besoin, les items manquants dans les listes Conjugaison / Morphologie / Syntaxe. Séparateur : saut de paragraphe.
  • Remarques : champ libre.
Tableur Grist (utilisateurs autorisés)

Accéder au tableur Grist →

  • Contributrice : indiquer le prénom et le nom du·de la contributeur·rice sous la forme Prénom Nom (majuscules à l'initiale).
  • Langue : préciser Latin ou Grec.
  • Auteur : renseigner le nom de l'auteur en français si existant. Sinon, renseigner le nom de l'auteur en latin.
  • Œuvre : renseigner le titre de l'œuvre en français si existant. Sinon, renseigner le nom de l'œuvre dans la langue source.
  • Référence : entrer les références du texte en chiffres arabes, séparateur point, sans p., § ou v.
  • Nombre de mots : indiquer le nombre de mots approximatif, sans mention « environ ».
  • Coupes : cocher la case si le texte a subi des coupes ; sinon, la laisser vide.
  • Niveau : suivre la documentation relative.
  • Conj / Morphologie / Syntaxe : renseigner les points remarquables via l'autocomplétion. Si un item manque, l'ajouter EN PREMIER LIEU dans les pages Conjugaison / Morphologie / Syntaxe accessibles depuis la marge gauche.
  • Texte latin / grec : renseigner le texte original. En prose, ne pas mettre de sauts de paragraphe ; en poésie, utiliser un saut de paragraphe entre les vers (sans séparateur /).
  • Traduction : renseigner la traduction. Même règles que pour le texte original (prose / poésie).
  • Crédit traduction : indiquer le prénom et le nom du·de la traducteur·rice sous la forme Prénom Nom.
